* **🚀 Strategic Growth for Future Roles (미래 지향적 역량 확장)**
  * **Full-stack Transition**: React, Next.js, TypeScript를 개인 프로젝트에 도입하여 백엔드 설계 단계부터 사용자 경험 (UX (User Experience))을 고려한 인터페이스 통합 능력을 배양하고 있습니다.

  * **Advanced OPS Engineering**: ArgoCD를 통한 GitOps (깃옵스) 자동화 환경을 구축하고, Kafka 기반의 EDA (Event-Driven Architecture (이벤트 중심 아키텍처))를 연구하여 데이터 스트리밍 역량을 확보하고 있습니다.
