<h2 align="center"> 🎯 Current Focus (요즘 집중하고 있는 것들) </h2>

<div>

실무에서 검증된 기술들을 더욱 날카롭게 다듬는 동시에, 다음 세대의 엔지니어링
역량을 확보하기 위해 아래 분야들에 몰입하고 있습니다.

* **🔨 인프라와 백엔드의 한계 돌파**
  * **Terraform**: NCP/Azure 인프라 프로비저닝을 100% 자동화했습니다. 선언적
관리를 통해 인프라의 가시성과 운영 수준을 한 단계 높였습니다.
  * **성능 최적화**: API 응답 속도 체감 수준을 끌어올리고, Redis와 Celery를
조합한 비동기 처리로 대규모 트래픽에도 흔들리지 않는 구조를 만듭니다.
  * **관측 가능성(Observability)**: Loki, Tempo, Alloy를 통합하여 장애 복구
시간(MTTR)을 30% 단축한 경험을 바탕으로, 엔지니어링 표준을 정립하고
있습니다.
  * **시스템 신뢰성**: PostgreSQL HA와 Redis Cluster 운영 노하우를 녹여내어,
어떤 상황에서도 스스로 살아나는(Self-healing) 자동 복구 시스템을 구현합니다.

* **🚀 기술적 도전과 탐구**
  * **풀스택으로의 확장**: React, Next.js, TypeScript를 도입해 백엔드 설계
단계부터 사용자 경험(UX)을 함께 고려하는 통합적인 능력을 키우고 있습니다.
  * **AI 워크플로우 혁신**: Ollama와 LangChain으로 나만의 지능형 에이전트를
구축하고, 실무 워크플로우에 AI를 수혈하여 생산성을 극대화하는 다양한 시도를
하고 있습니다.
  * **에이전틱 워크플로우(Agentic Workflows)**: LangGraph와 CrewAI를 활용해 단순 자동화를 넘어 스스로 판단하고 도구를 사용하는 자율형 에이전트 시스템을 설계합니다. 특히 MCP(Model Context Protocol)를 통해 에이전트 간 도구 공유의 표준화를 탐구하고 있습니다.
  * **고급 Ops 엔지니어링**: ArgoCD 기반의 GitOps 환경을 구축하고, Kafka를 활용한 이벤트 중심 아키텍처(EDA)를 연구하며 데이터 흐름의 주도권을 잡고 있습니다.
