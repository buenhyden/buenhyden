<h2 align="center"> 🎯 Current Focus (현재 집중 분야) </h2>

<div>

실무에서 검증된 기술을 고도화하는 동시에, 다음 단계의 엔지니어링 역량을 확보하기 위해 다음 분야들에 집중하고 있습니다.

* **🔨 인프라와 백엔드 최적화**
  * **Terraform**: NCP/Azure 인프라 프로비저닝을 100% 자동화하여 관리 가시성을 확보했습니다.
  * **성능 튜닝**: API 응답 속도를 개선하고, Redis와 Celery를 활용한 비동기 구조로 대규모 트래픽 대응력을 높입니다.
  * **Observability**: Loki, Tempo, Alloy를 통합하여 장애 복구 시간(MTTR)을 30% 단축하는 엔지니어링 표준을 정립 중입니다.
  * **신뢰성 확보**: PostgreSQL HA와 Redis Cluster 운영 경험을 바탕으로, 무중단 자동 복구 시스템을 구현합니다.

* **🚀 기술적 확장과 탐구**
  * **풀스택 전환**: React, Next.js, TypeScript를 도입해 사용자 경험을 고려한 인터페이스 통합 능력을 키우고 있습니다.
  * **AI/ML 워크플로**: Ollama와 LangChain으로 지능형 에이전트를 구축해 생산성을 높이는 PoC를 진행하고 있습니다.
  * **고급 OPS**: ArgoCD 기반의 GitOps 환경을 구축하고, Kafka를 이용한 이벤트 중심 아키텍처(EDA)를 연구합니다.
